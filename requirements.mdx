Alpha Piscium uses cutting-edge graphics technology that places real demands on both your GPU and its drivers. Before installing, confirm that your hardware and software meet the requirements below. GPU vendor support in particular is not uniform — one vendor is entirely unsupported, and another requires a specific minimum driver version to function correctly.

## Mod Loader & Shader Loader

Alpha Piscium is loaded through **Iris**, a dedicated shader loader for Minecraft. Iris version **1.7 or higher** is required. Earlier versions of Iris do not support the rendering features Alpha Piscium depends on and will fail to load the pack.

<Card title="Download Iris" icon="download" href="https://www.irisshaders.dev/download">
  Get Iris 1.7 or higher from the official Iris website. The installer sets up a ready-to-use Minecraft launcher profile.
</Card>

## GPU Requirements

<Warning>
  **Intel GPUs are not supported.** Alpha Piscium relies on graphics features that require a well-maintained OpenGL driver. Intel does not maintain their OpenGL driver to the standard required, and no workaround is available. If you have an Intel integrated or discrete GPU, Alpha Piscium will not run correctly.
</Warning>

<Accordion title="NVIDIA — Supported">
  Modern NVIDIA GPUs are fully supported and represent the primary development and testing target for Alpha Piscium. Any recent NVIDIA discrete GPU with a current driver should work without additional configuration.

  For best results, keep your NVIDIA driver up to date via GeForce Experience or the [NVIDIA driver download page](https://www.nvidia.com/en-us/drivers/).
</Accordion>

<Accordion title="AMD — Supported (Driver Version Required)">
  Modern AMD GPUs are supported, but your driver version must be **newer than 22.7.1**. Older AMD drivers contain OpenGL bugs that prevent Alpha Piscium from loading or rendering correctly.

  <Note>
    If you are on an AMD GPU and experiencing loading issues, open **AMD Software: Adrenalin Edition** (or download the latest driver from [AMD's driver page](https://www.amd.com/en/support)) and update to a driver version newer than 22.7.1 before attempting to run Alpha Piscium.
  </Note>

  To check your current driver version on Windows, open **Device Manager → Display Adapters**, right-click your AMD GPU, select **Properties**, and navigate to the **Driver** tab.
</Accordion>

<Accordion title="Intel — Not Supported">
  Intel GPUs are explicitly not supported. Alpha Piscium uses cutting-edge OpenGL features, and Intel does not maintain their OpenGL driver to the standard these features require. This applies to both Intel integrated graphics (e.g., Intel Iris Xe) and Intel Arc discrete GPUs.

  There is no configuration or workaround that enables Alpha Piscium to run on Intel hardware. A dedicated NVIDIA or AMD GPU is required.
</Accordion>

## Optional: Distant Horizons & Voxy

Alpha Piscium supports integration with **Distant Horizons** and **Voxy**, mods that extend Minecraft's view distance using Level of Detail (LOD) rendering. If you have either mod installed alongside Iris, Alpha Piscium will work with their extended terrain rendering.
